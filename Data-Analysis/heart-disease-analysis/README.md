# 🩺 Heart Disease UCI: Diagnostic Data Exploration

This project performs an in-depth Exploratory Data Analysis (EDA) on the UCI Heart Disease dataset. The objective is to identify the primary risk factors and correlations that contribute to cardiovascular diseases using statistical visualization.

---

## 📊 Clinical Insights & Analysis

The analysis focuses on several critical health indicators:

* **Demographic Impact:** Examining how age and sex correlate with the prevalence of heart disease.
* **Chest Pain (CP) Analysis:** Categorizing different types of chest pain and their relationship to heart health.
* **Physiological Factors:** Analyzing the impact of 'trestbps' (resting blood pressure) and 'chol' (serum cholesterol) on diagnosis.
* **Data Integrity:** Handled significant missing values in clinical features like `slope`, `ca`, and `thal` to ensure unbiased results.

---

## 🛠️ Data Pipeline & Tech Stack

* **Data Cleaning:** Implemented robust strategies for handling missing clinical records and data type conversion.
* **Statistical Visualization:** Used heatmaps to find correlations between health metrics and distribution plots for age-based trends.
* **Tech Used:** Python (Pandas, NumPy), Matplotlib, Seaborn.

---

## 🚀 Key Findings
1. Identified the age groups most vulnerable to specific types of chest pain.
2. Mapped the correlation between maximum heart rate achieved (thalch) and disease presence.
3. Visualized how physiological factors like cholesterol levels vary across different diagnostic datasets (Cleveland, Hungary, etc.).
