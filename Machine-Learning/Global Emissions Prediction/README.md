# Global Agrofood CO2 Emissions: Prediction & Interactive Analysis

This project delivers a high-performance machine learning solution to analyze and predict CO2 emissions within the global food system. By leveraging advanced regression techniques and interactive web deployment, it provides actionable insights into environmental impact trends.

---

## 🚀 Key Achievements

* **Top-Tier Performance:** Achieved an **R2 score of 0.992** using the **Random Forest Regressor**, surpassing existing public benchmarks on Kaggle for this specific dataset.
* **Robust Data Pipeline:** Led the preprocessing and feature engineering of a complex dataset comprising **7,000+ records and 31 features**, ensuring maximum model reliability and performance.
* **Interactive Deployment:** Developed a real-time **Web Dashboard via Streamlit**, allowing users to visualize global emission trends and generate custom predictions based on user-defined variables.
* **Advanced Visualizations:** Utilized **Matplotlib and Seaborn** to identify and present key factors contributing to global food system emissions through sophisticated data storytelling.

---

## 🛠️ Tech Stack

* **Language:** Python
* **Machine Learning:** Scikit-learn (Random Forest Regressor), Pandas, NumPy
* **Deployment:** Streamlit
* **Data Visualization:** Seaborn, Matplotlib

---

## 📊 Project Highlights

1.  **Exploratory Data Analysis (EDA):** Deep dive into 31 features to uncover correlations between agricultural practices and CO2 output.
2.  **Feature Engineering:** Strategic selection and transformation of variables to boost model accuracy to its peak.
3.  **Predictive Modeling:** Comparison of multiple regression models, with Random Forest emerging as the most effective.
4.  **User Interface:** A seamless Streamlit app that bridges the gap between complex data and end-user accessibility.
